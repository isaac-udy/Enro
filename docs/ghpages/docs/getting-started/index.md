# Getting Started

These three pages take you from an empty project to a working Enro screen,
in order.

- [Installation](installation.md) — add Enro to your project and install
  it on each platform.
- [Basic Concepts](basic-concepts.md) — the small vocabulary you need to
  read the rest of the docs.
- [Your First Screen](your-first-screen.md) — a complete worked example,
  from defining a key to receiving a result.

If you're upgrading an existing Enro 2 app, jump to the
[Migrating from Enro 2](../migrating-from-v2.md) guide first.
